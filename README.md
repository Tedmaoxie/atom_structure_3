# 原子结构与元素周期律教学可视化（atom_structure_3）

一个面向高中化学教学的交互式可视化页面，支持原子半径、电离能、电负性、金属性/非金属性等指标的热力图与折线图联动播放，帮助学生理解元素性质的周期性与规律性。

- 在线预览（部署完成后）：https://tedmaoxie.github.io/atom_structure_3/
- 本地运行：直接打开 `index.html` 即可（无需安装依赖）

## 功能特性
- 属性选择：在页面左上角选择展示的属性（原子半径、电离能、电负性、金/非金属性）
- 趋势辅助：支持横向递减、纵向递增、金属性对角线等趋势标识
- 播放控制：播放/暂停、速率调节、播放范围选择，联动图表高亮当前元素
- 显示选项：是否显示镧系/锕系、是否显示 3–12 副族
- 视图切换：热力图与折线图视图联动，支持多参数同时对比

## 快速开始
- 双击 `index.html`，即可在浏览器中打开本项目
- 建议使用最新版 Chrome/Edge/Firefox，获得更佳性能与兼容性

## 数据来源与单位规范
- 详细说明见 `docs/sources.md`
- 采用 IUPAC 官方口径为主，并统一单位（如电离能统一显示为 `kJ/mol`）
