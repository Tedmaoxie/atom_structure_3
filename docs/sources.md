# 元素周期表数据来源与单位规范

本项目所用的元素周期表数据遵循权威科学来源与统一单位规范，便于教学展示与可视化对比。

## 权威数据来源
- IUPAC 周期表与标准原子量（官方）：https://iupac.org/what-we-do/periodic-table-of-elements/
- IUPAC FAIR Cookbook（推荐获取路径，指导从权威数据库获取元素属性）：https://iupac.github.io/WFChemCookbook/datasources/pubchem_ptable.html
- NIST Chemistry WebBook（补充部分原子与分子数据查询）：https://webbook.nist.gov/chemistry/

> 说明：以上来源用于数据采集与交叉校验；在教学场景中，采用 IUPAC 口径为主，NIST 作为辅助查证。

## 数据字段与单位
- 原子序数（Z）：整数，1–118
- 元素符号（symbol）与名称（name）：IUPAC 标准命名
- 周期（period）与族（group）：IUPAC 标准排布
- 原子量（atomic_mass）：单位统一为原子质量单位（u），保留三位小数（教学展示）
- 电子排布（electron_configuration）：采用简写形式（如 `[Ne] 3s2 3p5`），来源按 IUPAC 推荐口径
- 原子半径（radius_pm）：采用“共价半径（单键）”，单位 pm；若来源为 Å，则换算为 ×100
- 第一电离能（ionization）：统一显示为 `kJ/mol`；若原始来源为 `eV/原子`，换算系数 `96.485 kJ/mol·eV⁻¹`
- 电负性（electronegativity）：Pauling 标度

## 金属性/非金属性（教学模型）
- 金属性指数（metallicity）：依据半径↑、电离能↓、电负性↓归一化合成；范围 [0,1]
- 非金属性指数（nonmetallicity）：定义为 `1 − metallicity`
- 注：二者为教学用模型化指标，并非直接来源于 IUPAC/NIST；用于直观呈现对角线趋势。

## 教学与使用说明
- 所有数值展示保留两位或三位小数，便于对比与讲解。
- 缺失或无定义的值以灰显显示，并在悬浮提示与详情面板中解释原因（如稀有气体电负性缺失）。
- 放射性/人造元素：在详情“备注”字段标注“放射性/人造：部分数据不适用或为理论值”，以避免误导性展示。

## 版本与维护
- 数据采集与更新遵循 IUPAC 最新发布与 NIST 官方页面。
- 单位转换口径：
  - 电离能：`eV/原子 → kJ/mol`，系数 `96.485 kJ/mol·eV⁻¹`，展示时统一保留两位小数
  - 半径：统一采用“共价半径（单键）”，单位 `pm`；若来源为 Å，换算 `×100`
  - 原子量：单位 `u`，展示保留三位小数（教学场景）
- 若发现数据差异或口径冲突，优先以 IUPAC 为准，并在代码与文档标注差异来源与处理方式。
