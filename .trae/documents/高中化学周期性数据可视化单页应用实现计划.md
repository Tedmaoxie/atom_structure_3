## 项目目标
- 构建可交互的完整周期表布局（7周期、18族，含镧系/锕系分栏）。
- 支持四种属性的可视化切换：`原子半径(pm)`、`第一电离能(kJ/mol)`、`电负性(Pauling)`、`金属性(定量化指标)`。
- 提供热力图颜色编码（红=高、蓝=低）、精确数值悬浮提示（含科学解释）、自动播放动画、趋势高亮（横向递减、纵向递增、金属性对角线）、异常元素标注。
- 纯 HTML/CSS/JavaScript，无第三方框架；完备数据验证；中文详尽注释，适配多屏。

## 数据来源与规范
- 数据主源：IUPAC FAIR Cookbook 推荐的 PubChem 元素数据集（包含 `AtomicRadius`、`IonizationEnergy`、`Electronegativity` 等字段）。
  - 参考：Accessing the Chemical Elements Data from PubChem — IUPAC FAIR Cookbook（https://iupac.github.io/WFChemCookbook/datasources/pubchem_ptable.html）。
- 单位与口径统一：
  - 原子半径：统一使用“共价半径（单键）”，单位 `pm`；若数据源给出为 Å，则乘以 100 转换为 `pm`。
  - 第一电离能：若数据源为 `eV/原子`，统一换算为 `kJ/mol`（系数 `96.485 kJ/mol·eV⁻¹`），并保留两位小数。
  - 电负性：采用 Pauling 标度；若数据缺失则在 UI 中灰显并在悬浮提示说明来源与缺失原因。
- 金属性定量指标（派生量）：
  - 设计归一化指数 `MetallicityIndex ∈ [0,1]`，综合考虑：`半径↑`、`电离能↓`、`电负性↓`。
  - 公式示例（可调整权重）：`MI = w_r·norm(radius) + w_ie·(1−norm(ionization)) + w_en·(1−norm(electronegativity))`，默认权重 `w_r=0.4`、`w_ie=0.35`、`w_en=0.25`。
  - 在 UI 中明确标注为“模型化指标”，并在帮助文稿中说明非 IUPAC 直接给出、但用于教学直观展示。
- 异常元素标注（示例）：
  - B、Be 的第一电离能相对邻近元素的异常；O 与 N 的电离能反常（电子成对与排布导致）；在悬浮提示中给出简明解释。
- 科学解释文案：每个属性准备 2–3 句简明解释，随 hover 展示（例：有效核电荷、屏蔽效应对趋势的影响）。

## 前端架构
- 单页结构：`index.html`（内含 `<style>` 与 `<script>`），不依赖任何外部框架。
- 模块划分：
  - `DataStore`：静态 JSON 数据嵌入；统一单位与小数；提供 `validateData()`。
  - `Renderer`：生成周期表格子，应用颜色映射，绘制趋势覆盖层与图例。
  - `Controls`：属性切换下拉、播放/暂停/速度、趋势开关。
  - `Tooltip & DetailPanel`：悬浮提示与点击弹出详细卡片。
  - `AnimationEngine`：自动播放路径、属性渐变过渡。
- 代码风格：ES6+，模块化函数；全部关键逻辑配中文注释（通俗易懂）。

## 周期表布局
- CSS Grid 方案：
  - 主表格 `grid-template-columns: repeat(18, minmax(2.8rem, 1fr))`，行对应 1–7 周期。
  - 镧系/锕系：独立两行，标注为“镧系/锕系”，并在移动端可折叠。
- 位置映射数据：元素对象含 `period`、`group`，通过 `grid-row`、`grid-column` 定位；空位渲染占位。
- 响应式策略：
  - 断点（示例）：`≥1200px` 桌面密集布局；`768–1199px` 缩小格子与字体；`<768px` 使用缩略布局 + 可缩放与拖动查看。
  - 字体与格子随容器宽度自适配；工具条在窄屏改为两行。

## 颜色编码与图例
- 连续热力图：蓝→红（低→高），HSL 插值：`h ∈ [220, 0]`，`s≈80%`，`l≈50%`。
- 归一化：针对当前所选属性，按有效元素数据的 `min–max` 线性缩放；可选支持分位数缩放以抗异常值。
- 图例：右上角小型条形渐变，标注当前属性名、最小/最大值（两位小数，含单位）。

## 交互与动画
- 顶部控制：
  - 属性选择下拉（4 项），趋势选择（横向递减/纵向递增/金属性对角线），动画控制（播放/暂停/速度滑条 0.5×–2×）。
- 悬浮提示（hover）：
  - 展示元素符号/名称/原子序数、当前属性值（两位小数+单位）、简要科学解释；若异常元素，追加“异常”徽标与解释。
- 点击详情：
  - 右侧/弹窗显示：四属性全部值、结构电子排布、族/周期信息、解释文本与来源链接。
- 自动播放动画：
  - 路径模式：按同周期从左到右或同族自上而下高亮；属性值渐变（淡入淡出/脉冲）。
  - 趋势覆盖：
    - 横向递减：切换到电离能/电负性时强调同周期右移上升现象（或为“半径横向递减”）。
    - 纵向递增：强调半径随周期增大；电离能/电负性随向下递减。
    - 金属性对角线：从右上到左下增强；通过 `MetallicityIndex` 显示斜向梯度。
- 性能优化：`requestAnimationFrame` 驱动，最小化重排；预计算颜色与归一化映射。

## 数据验证机制
- `validateData()`：
  - 结构校验：元素数=118；每项包含必要字段（`Z`、`symbol`、`period`、`group`）。
  - 类型与范围：原子半径 `>0` 且 `<400 pm`；第一电离能 `>0` 且 `<3000 kJ/mol`；电负性 `0–4.5`。
  - 单位归一：自动识别来源单位，转换后再入库；展示时统一保留两位小数。
  - 缺失与异常：标记/灰显缺失值；异常值进控制台报告与 UI 顶部“数据验证通过/存在警告”提示。

## 异常与科学解释文本
- 预置解释库（按属性）：
  - 原子半径：有效核电荷↑、屏蔽效应；周期横向递减、族纵向递增。
  - 第一电离能：轨道能级、电子成对引起异常（O vs N，B/Be）；随核电荷与半径变化。
  - 电负性：Pauling 基于键能；与有效核电荷、原子半径相关。
  - 金属性指数：定义与局限；与导电性/延展性相关但非严格物性。
- 异常元素列表示例：`B`、`Be`、`O`、`N`、`Al`（在特定比较中呈现偏差）；在详情面板集中展示。

## 无障碍与响应式
- ARIA：格子可聚焦，键盘左右上下导航；工具按钮具备标签。
- 高对比度模式：在设置中切换为色盲友好配色（红/蓝替换为 `viridis` 风格备选）。
- 移动端：触控放大/拖动浏览；工具条分组为两行。
